# ADR-002: Gemini Pro as Primary AI Provider

## Status

Accepted

## Decision

A.T.L.A.S. will use Gemini Pro as the primary AI provider.

## Context

Gemini Pro will be used as the main cognitive engine for reasoning, planning and interaction.

## Consequences

A.T.L.A.S. must include an AI provider abstraction layer to allow future provider replacement.
